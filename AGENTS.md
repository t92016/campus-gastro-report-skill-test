# 校園出現集體腸胃不適症狀通報（測試 skill 製作）

## 專案概述

這是一個**測試專案**。我在另一台電腦的模型上完成了一個「校園集體腸胃不適症狀通報網」，
最後將整個製作流程請 AI Agent Cloud 做成一個 **skill**，並 push 到我的 GitHub repo。

本專案的目的是：**在別台電腦上，用那個 skill 能不能做出差不多功能（或同樣功能）的通報網**。
簡單說就是驗證 skill 的可重現性與跨機可攜性。

## 技術栈

- Google Apps Script（GAS）
- HTML / CSS / JavaScript（前端表單與 UI）
- Google Sheets（後端資料儲存）
